# AKcelerate Brand Kit

## Structure

```
brand-kit/
├── logo/           # Logo files (SVG, PNG in various formats)
├── letterhead/     # Official letterhead templates (DOCX, PDF)
└── social/         # Social media post templates
```

## Logo Usage

- Minimum size: 120px wide for digital, 1 inch for print
- Clear space: equal to the height of the "A" in "AK" on all sides
- Never stretch, rotate, or recolor the logo
- Use `logo-full-dark.svg` on light backgrounds
- Use `logo-full-light.svg` / `logo-white.svg` on dark backgrounds
