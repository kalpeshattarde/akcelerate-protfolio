📝 How to Add Your Own Files

1. Add Files to Folders:
  • public/documents/ - PDFs, Word docs
  • public/images/ - Photos, logos, graphics
  • public/videos/ - Video files (or use YouTube)
  • public/downloads/ - Zip files, code, templates
2. Link to Files in HTML:
